# Uptown Stitch Upholstery Guy - Brand Guidelines

## Logo Overview

The Uptown Stitch Upholstery Guy logo combines traditional craftsmanship with modern design sensibility, reflecting the business's expertise in both auto and furniture upholstery.

## Logo Files

### Primary Logo
- **File**: `logo.svg`
- **Format**: SVG (scalable)
- **Usage**: Website header, business cards, signage
- **Dimensions**: 320x80px (scalable)

### Icon-Only Version
- **File**: `logo-icon.svg`
- **Format**: SVG (scalable)
- **Usage**: Favicon, social media profiles, app icons
- **Dimensions**: 64x64px (scalable)

### Horizontal Logo
- **File**: `logo-horizontal.svg`
- **Format**: SVG (scalable)
- **Usage**: Wide spaces, website navigation
- **Dimensions**: 280x60px (scalable)

### Vertical Logo
- **File**: `logo-vertical.svg`
- **Format**: SVG (scalable)
- **Usage**: Vertical spaces, social media posts
- **Dimensions**: 120x120px (scalable)

## Color Palette

### Primary Colors
- **Deep Charcoal**: `#1a1a1a` (Main text, needle)
- **Dark Navy**: `#2c3e50` (Secondary text)
- **Warm Gold**: `#d4a574` (Accent, stitching, needle eye)

### Supporting Colors
- **Light Gray**: `#666666` (Tagline, secondary elements)
- **White**: `#ffffff` (Background, light mode)

## Typography

### Primary Font
- **Family**: Arial, sans-serif (system fallback)
- **Weights**: 600 (semi-bold), 700 (bold)
- **Style**: Clean, modern, highly readable

### Hierarchy
- **Main Name**: 18px, weight 700
- **Secondary Name**: 16px, weight 600
- **Tagline**: 9px, weight 400, letter-spacing 0.5

## Logo Elements

### Stitch Icon
- **Needle**: Represents precision and craftsmanship
- **Stitch Lines**: Symbolize the core upholstery work
- **Seat Outline**: Subtle reference to furniture/auto upholstery

### Visual Meaning
- **Vertical Needle**: Professional expertise and precision
- **Horizontal Stitches**: Quality workmanship and attention to detail
- **Seat Silhouette**: Connection to both auto and furniture work

## Usage Guidelines

### Minimum Size
- **Primary Logo**: 120px width
- **Icon Only**: 16px (favicon), 32px (small icons)
- **Clear Space**: Minimum 10px around all sides

### Color Variations

#### Light Background (Recommended)
- Use default colors as specified
- Deep charcoal for main elements
- Gold accent for stitching

#### Dark Background
- Invert main colors to white
- Keep gold accent for stitching
- Maintain contrast ratios

#### Single Color
- Use deep charcoal only
- Maintain recognizability
- Suitable for engraving, single-color printing

### Do's
✅ Use on clean, uncluttered backgrounds
✅ Maintain aspect ratio and proportions
✅ Ensure sufficient contrast
✅ Use SVG format for web applications
✅ Provide clear space around logo

### Don'ts
❌ Stretch or distort proportions
❌ Add drop shadows or effects
❌ Change colors arbitrarily
❌ Place on busy/patterned backgrounds
❌ Use low-resolution versions

## Social Media

### Profile Pictures
- Use `logo-icon.svg` or `logo-vertical.svg`
- Ensure visibility at small sizes
- Maintain brand consistency

### Cover Photos
- Use `logo-horizontal.svg` with brand colors
- Include tagline when space permits
- Maintain professional appearance

## Print Specifications

### Business Cards
- Use `logo-horizontal.svg`
- Minimum 300 DPI resolution
- CMYK color conversion for printing

### Signage
- Use `logo-vertical.svg` for vertical signs
- Use `logo-horizontal.svg` for horizontal signs
- Ensure visibility from distance

## File Formats

### Vector (Recommended)
- **SVG**: For web, scalable applications

### Raster (When necessary)

- **ICO**: For favicon
